# 项目简介

基于深度学习的YOLOv8吸烟识别检测系统，利用先进的目标检测技术，实现对吸烟行为的实时识别与监控。系统包含YOLO数据集、用户界面和Python项目源码，以高精度识别公共场所吸烟行为，助力提升公共健康管理效率。技术特点包括快速响应、高检测准确率和易于部署，为构建无烟环境提供了有效的技术支持。


## 核心功能
- 基于深度学习YOLOv8的吸烟行为实时识别与监控
- 包含YOLO数据集，用于模型训练与优化
- 提供用户界面，便于操作与管理
- 完整的Python项目源码，支持二次开发
- 高精度识别公共场所吸烟行为
- 提升公共健康管理效率

## 适用场景
- 公共场所吸烟行为监控
- 无烟环境建设支持
- 公共健康管理辅助
- 安防监控系统中的特定行为识别

## 技术特点
- 采用先进的目标检测技术
- 快速响应能力
- 高检测准确率
- 易于部署
- 基于YOLOv8架构，支持实时处理
